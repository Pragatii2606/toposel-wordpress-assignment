

# Toposel WordPress Assignment

## Overview
This project is a mobile-first WordPress homepage built as part of the Toposel assignment.

## Tech Stack
- WordPress (Local setup)
- Advanced Custom Fields (ACF)
- WooCommerce
- Gutenberg Editor

## Features
- Mobile-only homepage layout
- Dynamic announcement bar
- Editable hero section (image, text, CTA)
- Dynamic brand logos (ACF)
- New Arrivals section powered by WooCommerce
- No hardcoded content

## CMS Usability
All homepage content can be managed via the WordPress admin panel using ACF fields. No technical knowledge is required for updates.

## Setup Instructions
1. Install WordPress locally
2. Copy the theme folder into:
   wp-content/themes/
3. Activate the theme
4. Import ACF field groups
5. Assign products to categories for New Arrivals

## Notes
This project focuses only on the mobile layout as per the assignment instructions.
